## Abstract

This project presents an AI-powered Personal Assistant capable of understanding and responding to user commands in natural language. The system integrates Generative AI with automation tools to perform tasks such as opening applications, launching websites, sending WhatsApp messages, drafting emails, managing contacts, and maintaining conversation memory.

The assistant is built using FastAPI for backend services and modern web technologies for the frontend. By utilizing Large Language Models (LLMs), the assistant can understand user intent and generate intelligent responses. The memory module stores previous conversations to provide a personalized experience. The project demonstrates how AI can simplify daily digital tasks and improve productivity through natural human-computer interaction.

## Introduction

Artificial Intelligence has transformed the way users interact with computers and digital devices. Traditional software requires users to navigate multiple menus and interfaces to perform tasks. An AI Personal Assistant simplifies this process by allowing users to communicate using natural language.

The objective of this project is to create a smart assistant capable of understanding user requests and performing various actions automatically. The system combines conversational AI, task automation, and memory management to provide a seamless user experience.

Main objectives:

* Understand natural language commands
* Automate routine computer tasks
* Open websites and applications
* Send WhatsApp messages and emails
* Store conversation history for personalization
* Provide intelligent responses using Generative AI

## Literature Review

Several AI assistants such as Siri, Google Assistant, Alexa, and ChatGPT have demonstrated the effectiveness of conversational AI in everyday applications. These systems utilize Natural Language Processing (NLP), Machine Learning, and Large Language Models to understand and respond to user queries.

Recent advancements in Generative AI have significantly improved contextual understanding and response generation. Open-source frameworks such as FastAPI and modern LLM APIs have enabled developers to build intelligent assistants with advanced capabilities. This project combines these technologies to create a personalized AI assistant capable of performing both conversational and automation tasks.

---

## Methodology (100 words)

The system receives user commands through a web-based chat interface. The input is processed using a Large Language Model to determine the user's intent. Based on the detected intent, the assistant performs actions such as opening applications, launching websites, sending WhatsApp messages, generating emails, or answering questions. A memory module stores previous interactions in JSON format to maintain conversation context. FastAPI serves as the backend framework that manages requests and communicates with external APIs when required. The frontend provides a user-friendly interface for interaction. The modular architecture ensures scalability, maintainability, and easy integration of future AI capabilities.
